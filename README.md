# Quants Projects Portfolio

A collection of quantitative analysis and machine learning projects focused on financial modeling and stock prediction.

## Projects

### 1. Portfolio Optimization - Mean Variance Optimization
Analysis of optimal portfolio allocation using mean-variance optimization techniques. Includes efficient frontier analysis and Sharpe ratio optimization.

### 2. ML Stock Prediction
Machine learning models for predicting stock price movements. Includes feature engineering, model training, and performance evaluation.

### 3. Volatility Forecast
Time series analysis and forecasting of stock market volatility. Includes GARCH models and other volatility prediction techniques.

### 4. Linear Regression Stock Return Prediction
Linear regression models for predicting stock returns based on various financial indicators and market factors.
